# Prog

Scraps web page and push notifications to subscribed users.

## Install

```bash
pnpm i
```

## Run

```bash
pnpm start
```

## Test

```bash
pnpm test
```

End-to-end tests: 

```bash
test:e2e
```

With coverage: 

```bash
pnpm test:cov
```

## Versions

- pnpm `v8.7.5`
- node `v20.9.0`
